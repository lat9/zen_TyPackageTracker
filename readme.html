Prerequisites

This version of Ty Package Tracker (TyPT) requires:

  1. A working copy of Zen Cart 2.1.0 or later. If you're using an older version of Zen Cart, please use the version of Ty Package Tracker that is associated with your Zen Cart's version.
  2. The site must use a PHP version greater than or equal to PHP 8.0.
  3. For integration with Edit Orders, use v5.0.0 or later of that plugin.

Purpose & Aim

This plugin adds the ability for your customers and admin to track packages from different carriers. The default carriers included are FedEx, UPS and USPS.

On your site's storefront:

  1. A separate sidebox is included that displays a logged-in customer's previous orders for quick reference.
  2. Customers can track their orders from the above sidebox or from their My Account page.

In your site's admin:

  1. An additional configuration group is created, where you can change, add or disable a carrier. Up to five (5) different carriers can be configured.
  2. An observer-class module provides entry of the package-tracking codes within either the built-in Customers :: Orders processing or via the Edit Orders plugin.

Installation

The installation steps for v5.0.0 and later of Ty Package Tracker are much simpler than those for prior versions of the plugin, since the plugin is now encapsulated. There are no longer any core-file or template-override edits required!

With this simplification, the upgrade from prior versions is a bit more complex, as the core-file and template changes must be backed-out and now-redundant files individually removed. See this GitHub Wiki article for additional information.

  1. Unzip the plugin's zip-file distribution.
  2. Copy the contents for the distribution's zc_plugins directory to your site's /zc_plugins directory.
  3. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  4. You'll see a plugin named Ty Package Tracker. Click on the associated Install button, confirm the installation and you're finished!
  5. Now that the plugin's installed, refer to this readme's How to Use tab to configure the plugin for your use.
Database Modifications

The plugin's admin installation script makes modifications to your site's database:

  1. A new configuration_group entry is created for the plugin's configuration settings.
  2. Additional configuration table entries are created for the plugin's configuration settings.
  3. The orders_status_history table is augmented with five (5) additional fields, associated with the five possible package carriers defined.
Uninstalling the Plugin

To uninstall the plugin:

  1. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  2. Locate the entry for Ty Package Tracker, click the associated Un-Install button, confirm the removal and you've removed the plugin.

How to Use

Configuring Ty Package Tracker

The Ty Package Tracker configuration settings are available via your Zen Cart admin's Configuration :: Ty Package Tracker. The following settings are available:

Name Description
The following three (3) settings are provided for Carriers 1 through 5.
Package Tracking - Carrier n Status Identifies whether (True) or not (False) the associated carrier is to be used on the site.
Package Tracking - Carrier n Name Identifies the name, e.g. FedEx, UPS and USPS, of the associated carrier.
Package Tracking - Carrier n Tracking Link Identifies the carrier-specific tracking link to which a tracking code is appended to view the package's tracking status.
Max display for Track Order sidebox Identifies the maximum number of orders to display in the Track Orders sidebox; default: 3.
Setting Orders' Tracking Numbers

You use your admin's Customers :: Orders tool's "Details" (provided by the Zen Cart core) or "Edit" (provided by the Edit Orders plugin) views to assign a carrier-specific tracking number for the order.

When you click the Update button, the value you enter is saved as part of the order's "status-history" and, if you've indicated that the status-update is to be visible to the customer, is viewable by the order's customer via their storefront Account History. The tracking number entered is appended to the carrier's configured tracking-link to display the package's tracking link in a separate browser window.

Adding or Updating Carriers

To add new carriers, you will need the carrier-specific tracking link. Navigate to the admin's Configuration :: Ty Package Tracker and, for each new carrier edit the Carrier Name and Carrier Tracking Link. Once those values are supplied, set the carrier's Status to True.

For example, if the URL for tracking is https://www.fedex.com/fedextrack/summary?trknbr=1234567890, you'll omit the tracking number (1234567890) and enter the Tracking Link as https://www.fedex.com/fedextrack/summary?trknbr=.

Additional Carrier URLs

The following additional carrier URLs have been supplied by the Zen Cart community:

  1. Canada Post URL: https://www.canadapost-postescanada.ca/track-reperage/en#/search?searchFor=. Shared by windsurfer in this support-thread posting and updated by torvista in this GitHub pull-request.
  2. SDA Express Courier URL: http://wwww.sda.it/SITO_SDA-WEB/dispatcher?invoker=home&LEN=&execute2=ActionTracking.doGetTrackingHome&button=Vai&id_ldv=. Shared by Colosports in this support-thread posting.
  3. Australia Post URL: https://auspost.com.au/track/track.html?id=. Shared by millsi in this support-thread posting.
  4. Royal Mail https://www.royalmail.com/portal/rm/track?trackNumber=. Provided by torvista in this GitHub pull-request.
